Cascading Style Sheets

  • CSS được sử dụng để mô tả định dạng, kiểu dáng (gọi chung là phong cách - style) của trang web.
  • CSS mô tả cách các phần tử HTML sẽ hiển thị trên màn hình, trang web, hoặc các phương tiện khác.

  • CSS giúp chúng ta tiết kiệm được rất nhiều công sức. Nó giúp chúng ta chỉnh sửa giao diện của nhiều trang web cùng một lúc.
  • Các mô tả css có thể được lưu trong tệp tin CSS.
